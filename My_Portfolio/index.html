Item list management

This application exhibit some data stored in a remote database, in a nicely designed table. That way, we can handle it with ease : add, update, delete, sort data. Every change of the is passed to the remote database by the mean of the server application, which is also deployed in the "Heroku" server. The application is secured using the "Basic Auth" and "JWT" technoligies.

Gestion d'une liste de données

Cette application présente des données issues d'une base de données distante, dans un tableau élégamment conçu. Chaque changement effectué sur les données du tableau est reflété par la suite dans la base de données via l'application serveur, qui est aussi hébergée dans le serveur d'Héroku. L'application est sécurisé via les technologies d'Authentification Basique et "JWT".

Surface Calculator

This application provide the user with an area where he can draw whatever shape he wants, either by using the mouse or by entering the coordinates of the dots making the shape. Once the drawing is done, then the application begin to calculate the surface of the shape. It shows some of the general steps of the process of calculation via animations.

Calcul de Surface

Cette application permet à l'utilisateur de dessiner une figure. une fois le dessin est terminé, elle se charge de calculer sa surface. Aussi, elle montre le processus générale du calcule via des animations.
